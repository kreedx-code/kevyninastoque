# Full Stack Developer Portfolio 

Este repositório contém o código-fonte de um portfólio profissional moderno, desenvolvido para apresentar habilidades, projetos e experiência na área de desenvolvimento de software.

O objetivo deste projeto é fornecer uma apresentação clara, organizada e profissional para recrutadores, empresas e clientes.

# Sobre o Projeto 

Este portfólio foi desenvolvido com foco em:

Apresentação profissional Design moderno Alta performance Boa experiência do usuário Organização e escalabilidade 

O site inclui recursos interativos e visuais utilizados em portfólios profissionais reais.

Tecnologias Utilizadas Front-end HTML5 CSS3 JavaScript Recursos e Funcionalidades Sistema de tema Dark e Light Animações suaves ao rolar a página Layout totalmente responsivo Sistema de sugestões integrado Salvamento automático de dados no navegador Interface moderna e profissional Hospedagem GitHub Pages Estrutura do Projeto portfolio/ │ ├── index.html ├── style.css ├── script.js └── README.md Funcionalidades Sistema de Tema 

Permite alternar entre modo escuro e claro. A preferência do usuário é salva automaticamente.

# Animações 

Elementos aparecem com transições suaves, proporcionando melhor experiência visual.

# Sistema de Sugestões 

Visitantes podem enviar sugestões diretamente pelo site.

Características:

Salvamento automático Persistência após recarregar a página Exibição pública das sugestões Atualização dinâmica Responsividade 

O site é compatível com:

Celulares Tablets Computadores Objetivo do Portfólio 

Este projeto foi criado para demonstrar competências em desenvolvimento Full Stack, incluindo:

Criação de interfaces modernas Organização de código Estruturação de projetos profissionais Desenvolvimento de sistemas web interativos Como Utilizar 1. Baixar ou clonar o projeto git clone https://github.com/seuusuario/seuusuario.github.io.git 

Ou baixar o arquivo ZIP.

2. Editar o conteúdo 

Abra o arquivo:

index.html 

E personalize com suas informações, projetos e descrição profissional.

3. Publicar no GitHub Pages 

Passos:

Criar um repositório chamado: seuusuario.github.io 

Enviar os arquivos do portfólio

Acessar:

Settings → Pages Ativar o GitHub Pages 

O site ficará disponível publicamente.
